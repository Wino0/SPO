lexerCl.py  Разбивает входные строки из input.txt на лексемы. 
parserCl.py  Принимает лексемы от Лексера, сторит дерево, "причесывающее" код.
stackMacCl.py  Принимает дерево от Парсера, записывая его в ПОЛИЗ.

Для работы необходим текстовый докумет в формате input.txt, содержащий входной код.

Могут быть проблемы с юникодом, если прогонять через дебаг. Фиксится либо big brain либо несколько раз перезапустить дебаг.
